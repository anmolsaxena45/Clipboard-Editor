## A basic Clipboard-Editor (React + Redux)

A modern, intuitive web application built with React, Redux Toolkit, Tailwind CSS, and Heroicons that allows users to create, view, search, copy, share, and delete text snippets (pastes). Designed for productivity and simplicity, this app provides a clean interface with powerful state management.

# Features
Search Functionality  
Quickly filter pastes by title using the search bar.

CRUD Operations

1. Edit: Navigate to edit an existing paste.

2. View: Open a detailed view of the paste.

3. Delete: Remove unwanted pastes with a single click.

4. Copy: Instantly copy paste content to your clipboard with toast notifications.

5. Share: Share paste content directly via WhatsApp.

# Responsive UI  
Built with Tailwind CSS, ensuring a sleek and responsive design.

State Management  
Powered by Redux Toolkit, making paste operations predictable and efficient.

User Feedback  
Integrated with React Toastify for smooth, non-intrusive notifications.

# Tech Stack
Frontend: React, Tailwind CSS

State Management: Redux Toolkit

Icons: Heroicons

Notifications: React Toastify
